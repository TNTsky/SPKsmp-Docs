# 伺服器規則

## 關於可複製物品

一律由管理員複製。任何交易可得或是合成可得或怪物掉落物皆不可複製。

### 可複製名單 (僅使用原版複製方法)

- 海綿
- 海靈核心
- 非法方塊（如基岩、地獄入口、氣泡柱等）
- 鞘翅

（我想到再加）

若被抓到非法複製將全數沒收銷毀。

## 禁止外掛刷物品

Toolbox 類等外掛僅允許靈魂出竅、放大、修改 gamma 值等輔助功能。

## 請勿濫用回檔功能

若非公共設施重大損壞，不要找我回檔。

## 請勿惡意破壞

違者直接踢出。其餘違規視情況給予處分。
